## Description

Projeto simples com 3 entidades (Users, Orders e Products) para treinar as seguintes ferramentas:

- Mongoose
- MongoDB
- CASL
- Cron
- Multer
- Cookie-parser

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# docker
$ docker-compose up
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
